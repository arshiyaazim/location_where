# Employee Monitoring Application (Location Where)

একটি শক্তিশালী এবং সুরক্ষিত কর্মী পর্যবেক্ষণ সমাধান যা ফিনান্সিয়াল কোম্পানিগুলোর জন্য তৈরি করা হয়েছে। এটি কর্মীদের রিয়েল-টাইম লোকেশন, সিম কার্ডের নিরাপত্তা, এবং ডিভাইস ম্যানেজমেন্ট নিশ্চিত করে।

---

## 🏗 প্রজেক্ট স্ট্রাকচার (Project Structure)

প্রজেক্টটি তিনটি প্রধান ভাগে বিভক্ত:

1.  **/backend**: Node.js (Express) এপিআই, PostgreSQL ডাটাবেস এবং Redis ক্যাশ।
2.  **/app**: নেটিভ অ্যান্ড্রয়েড অ্যাপ্লিকেশন (Kotlin) যা কর্মীদের ডিভাইসে ইন্সটল থাকে।
3.  **/admin-dashboard**: রিঅ্যাক্ট (React) ভিত্তিক ওয়েব প্যানেল যেখান থেকে এডমিন সব কিছু নিয়ন্ত্রণ করেন।

---

## ✨ প্রধান বৈশিষ্ট্যসমূহ (Key Features)

### ✅ এখন পর্যন্ত যা তৈরি করা হয়েছে (Implemented):
*   **GPS লোকেশন ট্র্যাকিং:** প্রতি ৩০ সেকেন্ডে রিয়েল-টাইম লোকেশন আপডেট (অফলাইন মোড এবং রুম ডাটাবেস সাপোর্ট সহ)।
*   **জিওফেন্সিং (Geofencing):** নির্ধারিত এলাকার বাইরে গেলে স্বয়ংক্রিয় অ্যালার্ট এবং সার্ভারে রিপোর্ট।
*   **সিম পরিবর্তন সনাক্তকরণ:** ডিভাইসে নতুন সিম কার্ড লাগানো হলে তাৎক্ষণিক এডমিনকে জানানো এবং ডিভাইস লক কমান্ড।
*   **রিমোট কন্ট্রোল:** এডমিন প্যানেল থেকে সরাসরি ডিভাইস লক, সাইরেন বাজানো বা ডেটা মুছে ফেলার সুবিধা।
*   **ব্যাটারি অপ্টিমাইজেশন:** ব্যাটারি ৩০% এর নিচে নামলে স্বয়ংক্রিয়ভাবে ইন্টারভাল অ্যাডজাস্টমেন্ট।
*   **JWT অথেন্টিকেশন:** আধুনিক এক্সেস এবং রিফ্রেশ টোকেন ভিত্তিক সিকিউরিটি সিস্টেম।
*   **লাইভ ম্যাপ:** এডমিন ড্যাশবোর্ডে সব কর্মীর অবস্থান একসাথে দেখার সুবিধা।

### ⏳ যা এখনও বাকি (Pending / Roadmap):
*   **কল রেকর্ডিং (Call Recording):** অ্যান্ড্রয়েড অ্যাপ থেকে স্বয়ংক্রিয় কল রেকর্ড এবং এনক্রিপ্টেড ফাইল AWS S3-এ আপলোড।
*   **ওয়ার্ক ফোল্ডার মনিটরিং:** কোম্পানির নির্দিষ্ট ফোল্ডারে ফাইল শেয়ারিং এবং এক্সেস ট্র্যাকিং।
*   **বিস্তারিত রিপোর্ট:** সাপ্তাহিক এবং মাসিক পারফরম্যান্স PDF/Excel জেনারেটর।
*   **Socket.io ইন্টিগ্রেশন:** পোলিংয়ের পরিবর্তে আরও দ্রুত রিয়েল-টাইম কমিউনিকেশন।

## 🛠 টেকনোলজি স্ট্যাক (Tech Stack)

*   **Backend:** Node.js v20, Express, Prisma ORM, PostgreSQL 16, Redis 7, JWT, Zod.
*   **Android App:** Kotlin, Retrofit, Room DB, Hilt (DI), WorkManager, DataStore, Geofencing API.
*   **Dashboard:** React 18, Ant Design 5, TanStack Query, Leaflet Maps, Zustand.
*   **Cloud:** AWS S3 (Storage), Firebase Admin (Push Notification).

## 🚀 ইন্সটলেশন গাইড (Installation Guide)

### ১. ব্যাকএন্ড (Backend)
```bash
cd backend
npm install
cp .env.example .env  # আপনার ক্রেডেনশিয়াল সেট করুন
npx prisma migrate dev
npm run dev
```

### ২. এডমিন ড্যাশবোর্ড (Web Dashboard)
```bash
cd admin-dashboard
npm install
npm run dev
```

### ৩. অ্যান্ড্রয়েড অ্যাপ (Android App)
*   অ্যান্ড্রয়েড স্টুডিওতে `Location_Where` প্রজেক্টটি ওপেন করুন।
*   `RetrofitClient.kt` ফাইলে আপনার সার্ভার আইপি সেট করুন।
*   অ্যাপটি বিল্ড করে ফিজিক্যাল ডিভাইসে রান করুন (GPS পারমিশন নিশ্চিত করুন)।

## ☁️ ভিপিএস সার্ভার ডিপ্লয়মেন্ট (VPS Deployment Guide)

যদি আপনি একটি লিনাক্স (Ubuntu) ভিপিএস ব্যবহার করেন:

### ১. এনভায়রনমেন্ট সেটআপ:
```bash
sudo apt update
sudo apt install nodejs npm postgresql redis-server nginx
```

### ২. ডাটাবেস কনফিগারেশন:
PostgreSQL-এ একটি ডাটাবেস তৈরি করুন এবং `.env` ফাইলে তার লিঙ্ক দিন।

### ৩. সার্ভার চালু করা (PM2):
সার্ভারটি সবসময় সচল রাখতে PM2 ব্যবহার করুন:
```bash
sudo npm install -g pm2
cd backend
pm2 start dist/app.js --name "monitoring-api"
```

### ৪. এনজিনেক্স (Nginx) কনফিগারেশন:
আপনার ডোমেইন বা আইপি-তে ট্রাফিক প্রবাহিত করতে Nginx রিভার্স প্রক্সি ব্যবহার করুন:
```nginx
server {
    listen 80;
    server_name your-domain.com;
    location /api/ {
        proxy_pass http://localhost:3000;
    }
    location / {
        root /var/www/admin-dashboard/dist;
        try_files $uri /index.html;
    }
}
```

## 🔒 নিরাপত্তা (Security)
*   সব ডেটা **TLS 1.3** এর মাধ্যমে এনক্রিপ্টেড।
*   কল রেকর্ডিং **AES-256-CBC** ব্যবহার করে ফাইল লেভেলে সুরক্ষিত।
*   রোল বেজড এক্সেস কন্ট্রোল (RBAC) নিশ্চিত করা হয়েছে।
